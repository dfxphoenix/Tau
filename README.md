# DiscordMusicBot
<p>A complete sourcecode from Discord Music Bot written in JavaScript with NodeJS.</p>
<br>
<b>⚡ Commands</b>
<p>help - show list of commands</p>
<p>ping - get ping of bot</p>
<p>back - back the music</p>
<p>clear - clear playlist</p>
<p>filter - set filters to music</p>
<p>loop - repet current playlist</p>
<p>nowplaying - show current music</p>
<p>pause - pause current music</p>
<p>play - play a music</p>
<p>progress - show current playing time of current music</p>
<p>queue - show the playlist</p>
<p>resume - resume current music</p>
<p>save - send in DM the music you listened to</p>
<p>search - search a music</p>
<p>seek - seeking a music</p>
<p>shuffle - queue shuffled one music</p>
<p>skip - skipping current music</<</p>
<p>stop - stop current music</p>
<p>volume - set volume of music</p>
<br>
<b>📑 Instalation</b>
<p>Install NodeJS 16+</p>
<p>Open the configuration file (config.js) and edit all fields</p>
<p>Execute command <b>npm install</b> in bot directory</p>
<p>Start bot with command <b>npm start</b> in bot directory</p>
